# Ariana Version One Plan

Ariana version one is the stable API milestone after 0.5.0.

## Patch plan

| Patch | Area | Goal | Status |
| --- | --- | --- | --- |
| patch-01 | API governance | API lock and compatibility checks | Started |
| patch-02 | Runtime | Lifecycle and cleanup completion | Pending |
| patch-03 | Compiler | Template diagnostics and typecheck completion | Pending |
| patch-04 | Vite plugin | Resource transform and tests | Pending |
| patch-05 | Router | Nested routes, guards, lazy loading, navigation state | Pending |
| patch-06 | Forms | FormGroup, FormArray, validators, typed errors | Pending |
| patch-07 | Query | Mutation lifecycle, invalidation, stale policy, retries | Pending |
| patch-08 | Rendering | Server rendering contracts and render benchmarks | Pending |
| patch-09 | CLI | Buildable starters and package manager detection | Pending |
| patch-10 | Tests | Unit, integration, generated app, browser, package and type tests | Pending |
| patch-11 | Performance | Framework comparison, DOM benchmarks, bundle budgets | Pending |
| patch-12 | Docs | Public docs, migration guide, examples and recipes | Pending |
| patch-13 | Release | Changelog, release candidates and rollback notes | Pending |
| patch-14 | Readiness | Support matrix and known limitations | Pending |

## Exit criteria

- stable public API list exists
- compatibility check exists
- generated app builds and runs
- package install tests pass
- browser interaction tests pass
- framework benchmark output is captured
- migration guide from 0.5 to version one exists
