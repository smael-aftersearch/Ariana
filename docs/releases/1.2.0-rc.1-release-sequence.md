# Ariana 1.2.0-rc.1 Release Sequence

This is the recommended order for releasing Ariana `1.2.0-rc.1`.

## 1. Pull latest main

```bash
git pull origin main
npm install --no-audit --no-fund
```

## 2. Run local gates

```bash
npm run release:gates:v1.2
```

## 3. Run manual UI checks

```bash
npm run demo:admin -- --force
```

Open:

```txt
http://localhost:5173/animate
http://localhost:5173/route-outlet
```

## 4. Build GitHub candidate artifacts

Run workflow:

```txt
Actions -> Ariana 1.2 RC Release Candidate
```

Inputs:

```txt
version: 1.2.0-rc.1
run_gates: true
```

Expected result:

- gates pass;
- packed candidate smoke passes;
- release manifest is generated;
- tarballs are uploaded;
- manifest and sha256 checksum files are uploaded;
- release notes are uploaded.

## 5. Verify candidate artifact checksums

Download these workflow artifacts:

```txt
ariana-1.2.0-rc.1-npm-tarballs
ariana-1.2.0-rc.1-manifest
```

Place the tarballs and `ariana-1.2.0-rc.1-sha256.txt` in the same directory and run:

```bash
sha256sum -c ariana-1.2.0-rc.1-sha256.txt
```

Expected result: every tarball reports `OK`.

## 6. Run npm dry run

Run workflow:

```txt
Actions -> Ariana 1.2 NPM Dry Run
```

Inputs:

```txt
version: 1.2.0-rc.1
dist_tag: next
```

Expected result:

- gates pass;
- packed candidate smoke passes;
- npm dry run succeeds;
- no real package is published.

## 7. Create GitHub release draft

Run workflow:

```txt
Actions -> Ariana 1.2 GitHub Release
```

Inputs:

```txt
version: 1.2.0-rc.1
confirm: CREATE_GITHUB_RELEASE
```

Expected result:

- tag `v1.2.0-rc.1` is created;
- GitHub release is created as draft;
- release is marked prerelease;
- release notes are loaded from `docs/releases/1.2.0-rc.1.md`;
- package tarballs are attached;
- manifest and sha256 checksum files are attached.

## 8. Publish npm packages to next

Before this step, make sure:

- GitHub Environment `npm-next` exists;
- `npm-next` requires approval;
- repository secret `NPM_TOKEN` exists;
- previous workflows passed.

Run workflow:

```txt
Actions -> Ariana 1.2 Next Release
```

Inputs:

```txt
version: 1.2.0-rc.1
confirm: RELEASE_NEXT
```

Expected result:

- release gates pass;
- npm packages publish using `next` dist tag;
- tarballs are uploaded as artifacts.

## 9. Verify npm registry

Run workflow:

```txt
Actions -> Ariana 1.2 Verify NPM Release
```

Inputs:

```txt
version: 1.2.0-rc.1
dist_tag: next
```

Expected result:

- every `@ariana-framework/*` package exists at `1.2.0-rc.1`;
- every package has `next` pointing to `1.2.0-rc.1`.

## 10. External install smoke

Create a clean directory outside the repo:

```bash
mkdir ariana-rc-smoke
cd ariana-rc-smoke
npm init -y
npm install @ariana-framework/core@next @ariana-framework/router@next @ariana-framework/vite-plugin@next
```

Check that installed versions resolve to `1.2.0-rc.1`.

## 11. Publish GitHub draft release

After npm verification and external install smoke pass, manually review and publish the GitHub draft release.

Keep it marked as prerelease.
