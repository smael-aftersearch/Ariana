# Ariana Todo App Example

This is the first realistic Ariana example fixture after the 0.4.1 public release line.

## Purpose

The example checks that these packages can be used together:

- `@ariana-framework/core`
- `@ariana-framework/router`
- `@ariana-framework/forms`
- `@ariana-framework/query`

## Current status

This is a TypeScript fixture, not yet a full browser demo.

It should evolve into a real Vite app during the 0.5.0 milestone.

## Run

```bash
npm --prefix examples/todo-app install
npm --prefix examples/todo-app run build
```
