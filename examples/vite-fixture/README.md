# Ariana Vite Fixture

This fixture validates that Ariana packages can participate in a real Vite build.

## Packages covered

- `@ariana-framework/core`
- `@ariana-framework/vite-plugin`

## Run

```bash
npm --prefix examples/vite-fixture install
npm --prefix examples/vite-fixture run build
```

## Status

This is the first Vite integration fixture. It should grow into a component-template integration test during the 0.5.0 milestone.
