# create-ariana

Project starter for Ariana applications.

## Status

Early CLI skeleton for the 0.5.0 milestone.

## Usage

```bash
npm create ariana my-app
```

Generated app includes:

- Vite
- TypeScript
- `@ariana-framework/core`
- `@ariana-framework/vite-plugin`
