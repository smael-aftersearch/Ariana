# @ariana-framework/rendering

Rendering helpers for Ariana SSR, SSG, and island experiments.

## Includes

- HTML shell rendering
- static route page generation
- island definitions
- island placeholders
- island manifest script generation
- island activation and cleanup helpers

## Install

```bash
npm install @ariana-framework/rendering
```

## Status

Official early release. SSR/SSG/islands are not production-ready yet.
