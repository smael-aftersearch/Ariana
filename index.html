What changed for 1.1

Version 1.1 focuses on release confidence: faster reactive paths, safer core packaging, and examples that show real UI structure.

Compiled rendering only

Core no longer ships the legacy runtime template evaluator. Components use compiled render functions generated by the compiler and Vite plugin.

Performance guardrails

Reactivity, computed signals, keyed lists, and form arrays were optimized while keeping contract tests around lifecycle and snapshots.

Security release gates

The release path includes static security audit, package tarball inspection, npm dry-run, smoke tests, and manual GitHub Actions publishing.
